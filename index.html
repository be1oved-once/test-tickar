PRO EXAM SYSTEM

Temporary Live Tests with Built-in Cheating Control

Beforexam temporary tests are not just timed - they are actively monitored. Every action that breaks exam discipline is detected and punished automatically.

Admin Creates Live Test

Admin sets custom questions (MCQ or written), custom duration, and publishes the test instantly. Students see the Temp Test option immediately.

Active Monitoring Starts

The moment a student enters the test, the system starts tracking tab focus, app switching, visibility change, and split-screen behaviour.

Cheating = Instant Penalty

Leaving the tab, opening another app, switching windows, or backgrounding the test immediately locks the attempt with a penalty.

Admin Sees Everything

Admin opens the leaderboard to see ranks, marks, violations, and detailed answers - all in one place.

No refresh tricks. No multiple tabs. The test behaves like a real examination hall.
